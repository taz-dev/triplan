  * FrontEnd : 최은석
  * BackEnd : 박정현
  * UI/UX 디자이너 : 김종표

### ⚙ 기술 스택
* FrontEnd : React Native, Node.js
* BackEnd : JAVA(11), SpringBoot, Spring Security, Spring Data JPA, MariaDB
